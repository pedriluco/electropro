El Módulo de Electrotecnia
// Hardware físico — Ecosistema IPN de Aprendizaje Híbrido
🖥️
Módulo Físico de Electrotecnia
Plataforma de aprendizaje práctico que conecta la teoría eléctrica con la implementación real. Incluye Arduino UNO, panel de LEDs, fuente de alimentación, resistencias y conectores integrados.

Especificaciones

MicrocontroladorArduino UNO
Voltaje operación5V / 12V
Pines digitales14 (6 PWM)
Pines analógicos6
ConectividadUSB-B / UART

Arduino UNO
// Cerebro del módulo — documentación técnica
🤖
Arduino UNO R3
Microcontrolador ATmega328P. Integrado en el módulo físico del IPN para prácticas de circuitos digitales y analógicos.
// Pines PWM: 3, 5, 6, 9, 10, 11
// Analógicos: A0–A5
// Comunicación: UART, SPI, I2C
// Voltaje lógico: 5V
// Corriente máx. por pin: 40 mA

Ejemplo — Blink básico

// Primer programa: parpadeo del LED integrado
void setup() {
pinMode(13, OUTPUT);
}
void loop() {
digitalWrite(13, HIGH);
delay(1000);
digitalWrite(13, LOW);
delay(1000);
}
